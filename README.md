# Tugas1_Kriptografi

Note: Perlu XAMPP

Langkah Run Aplikasi:
Jalankan XAMPP Control Panel
Start Apache dan MySQL

>> Set DB
Buka browser, akses: http://localhost/phpmyadmin
Klik tab "SQL"
Copy seluruh isi file dumpp.sql yang telah saya berikan
Paste ke SQL editor di phpMyAdmin
Klik "Go" atau "Execute"

>> Set Website
Buka folder XAMPP (C:\xampp atau lainnya)
Open Folder htdocs
Buat folder baru: asset-management //ga terlalu masalah untuk nama folder, tp untuk mempermudah
Copy file index.html dan api.php ke folder htdocs/asset-management/
